#  Sales Performance Dashboard

##  Project Overview

This project presents an interactive **Sales Performance Dashboard** developed in **Microsoft Excel** using **Pivot Tables, Pivot Charts, Slicers, and Power Pivot**. The dashboard enables users to analyze sales performance across different territories, product categories, and brands through dynamic visualizations.

---

## Objectives

- Monitor overall sales performance
- Compare sales across territories and brands
- Analyze category-wise sales contribution
- Track daily sales trends
- Enable interactive filtering using slicers
- Support business decision-making through visual analytics
