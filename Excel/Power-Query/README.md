# Data Cleaning and Transformation with Excel Power Query

## Project Overview

This project demonstrates the use of Microsoft Excel Power Query to import, clean, transform, and prepare raw data for business analysis. Power Query was used to automate data preparation, ensuring consistency, accuracy, and efficiency before reporting.

---

## Business Objective

The objective of this project is to automate the Extract, Transform, and Load (ETL) process, reducing manual effort and preparing high-quality data for analysis and dashboard creation.

---

## Dataset

The dataset contains customer and transaction information, including:

- Customer ID
- Representative
- Duration
- Purchase Amount
- Order Date
- Sales Records

---

## Data Transformation Performed

The following Power Query transformations were applied:

- Imported data from Excel worksheets
- Removed duplicate records
- Changed data types
- Renamed columns
- Filtered unnecessary records
- Cleaned inconsistent values
- Sorted data
- Removed blank rows
- Prepared data for reporting
- Loaded cleaned data into Excel
