# Sales Analysis Using Excel Pivot Tables

## Project Overview

This project demonstrates the use of Microsoft Excel Pivot Tables, Pivot Charts, and Slicers to analyze sales performance across different territories, brands, categories, and time periods. The objective is to transform raw sales data into meaningful business insights for decision-making.
## Business Problem

A distribution company wants to evaluate sales performance by territory, product category, brand, and delivery status. Management requires interactive reports to monitor delivered amounts, order values, bounced values, and regional performance.
## Dataset

The dataset contains sales transaction information, including:

- Order Date
- Territory
- Area
- Category
- Brand
- Order Amount (TP)
- Delivered Amount (TP)
## Project Objectives

The analysis answers the following business questions:

1. What is the delivered amount by territory?
2. What is each territory's contribution to total delivered sales?
3. Which product categories generate the highest delivered amount?
4. Which brands perform best across different territories?
5. How does the Diploma brand perform compared to other brands?
6. How do order amounts compare with delivered amounts by brand?
7. Which brands have the highest bounced value (Order Amount − Delivered Amount)?
8. Which brands generate the highest sales?
9. Which areas contribute the most to total delivered sales?
10. How do delivered sales change over time?
