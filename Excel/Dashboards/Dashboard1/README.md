# 📈 Sales Performance Dashboard (Excel)

## Project Overview

This interactive Excel dashboard monitors sales team performance using key performance indicators (KPIs), monthly deal analysis, and regional service-level reporting.

---

## Business Problem

Sales managers need a centralized dashboard to monitor call performance, deal conversion, revenue generation, and regional service quality.

---

## Dashboard KPIs

- Total Calls
- Calls Reached
- Deals Closed
- Deal Value

---

## Dashboard Features

- KPI Cards
- Employee Performance Table
- Monthly Deals Analysis
- Service Level by Division
- Interactive Employee Filter

---

## Excel Skills Used

- Pivot Tables
- Pivot Charts
- Slicers
- Maps
- Conditional Formatting
- Dashboard Design

---

## Business Insights

- Identified top-performing sales representatives.
- Monitored monthly deal trends.
- Compared service levels across divisions.
- Evaluated overall sales performance.
