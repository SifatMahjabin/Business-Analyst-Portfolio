#  HR Analytics Dashboard (Excel)

## Project Overview

An HR dashboard developed in Microsoft Excel to analyze workforce demographics, salaries, education, employee satisfaction, and work-life balance.

---

## Business Problem

HR managers require a dashboard to monitor workforce metrics and support strategic human resource planning.

---

## Dashboard KPIs

- Total Employees
- Total Salary

---

## Dashboard Features

- Employee Distribution
- Education Analysis
- Job Satisfaction
- Work-Life Balance
- Salary Trend
- Experience Analysis
- Gender Analysis
- Department Filter

---

## Excel Skills Used

- Pivot Tables
- Pivot Charts
- Slicers
- KPI Cards
- Dashboard Design

---

## Business Insights

- Workforce demographics were clearly visualized.
- Education levels varied across employees.
- Job satisfaction remained relatively high.
- Salary trends changed throughout the year.

---

## Tools

Microsoft Excel
