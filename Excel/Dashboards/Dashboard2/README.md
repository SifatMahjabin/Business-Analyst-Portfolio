#  Nigerian Car Market Dashboard (Excel)

## Project Overview

An interactive dashboard analyzing Nigeria's automobile market using pricing, mileage, horsepower, transmission type, engine size, and regional distribution.

---

## Business Problem

Automobile dealers need insights into vehicle pricing, customer preferences, and regional demand to improve inventory planning.

---

## Dashboard KPIs

- Average Price
- Mileage
- Horsepower
- Engine Size

---

## Dashboard Features

- Vehicle Price Analysis
- Top Car Brands
- Transmission Preference
- Vehicle Color Analysis
- Nigeria Regional Map

---

## Excel Skills Used

- Pivot Tables
- Dashboard Design
- Maps
- Charts
- Interactive Filters

---

## Business Insights

- Toyota was among the most popular brands.
- Automatic transmission dominated customer preference.
- Regional demand varied across Nigeria.
- Vehicle colors showed distinct customer preferences.
