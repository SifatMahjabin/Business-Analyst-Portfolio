#  Microsoft Excel Data Analysis Projects

##  Overview

This repository contains a collection of Microsoft Excel projects focused on **data analysis, business intelligence, and interactive dashboard development**. The projects demonstrate the use of Excel's advanced analytical tools to transform raw data into meaningful insights through dashboard, reports, and visualizations.

---

##  Objectives

- Analyze business and sales data
- Build interactive dashboards
- Perform data cleaning and transformation
- Generate business insights using Pivot Tables and Charts
- Create dynamic reports for decision-making
