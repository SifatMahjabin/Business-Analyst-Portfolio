
## Project Overview
This project analyzes sales transaction data using SQL to uncover business insights related to customers, products, sales performance, and regional trends. The analysis demonstrates how SQL can be used to answer business questions and support data-driven decision-making.

## Dataset
The project uses three relational tables: Customers, Products, Sales

## Objectives

This project answers the following business questions:

- Which products generate the highest sales revenue?
- Which product categories perform best?
- Which regions purchase the most products?
- Which customers spend the most?
- How do sales change over time?
## SQL Skills Demonstrated

- SELECT
- WHERE
- ORDER BY
- GROUP BY
- HAVING
- INNER JOIN
- Aggregate Functions (SUM, COUNT, AVG)
- Date Functions
- Aliases
## Key Insights

- Electronics products generated the highest revenue.
- Customers from Dhaka purchased the highest-value products.
- Furniture sales contributed significantly through larger quantities.
- Revenue was driven by a small number of high-value transactions.
- Sales occurred across multiple months, allowing monthly performance tracking.
