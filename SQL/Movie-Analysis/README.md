# Movie Analysis
This project analyzes a movie database using SQL.
## Objective
Analyze a small movie dataset to demonstrate SQL querying and business analysis techniques.
## Dataset
- 10 movie records
- Fields: MovieID, Genre, Rating, Revenue, Release Year
## Business Questions
- Which genre has the highest average rating?
- Which movies generated the highest revenue?
- How many movies were released each year?
## SQL Concepts Used
- SELECT
- WHERE
- GROUP BY
- ORDER BY
## Tools Used

- MySQL
- SQL Workbench
